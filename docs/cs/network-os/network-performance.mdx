# Network Performance Optimization

## Core Metrics

- Latency (p50, p95, p99)
- Throughput
- Packet loss
- Retransmission rate

## Optimization Pattern

1. Define SLO.
2. Measure baseline.
3. Change one factor.
4. Re-measure and compare.
