# Linux Performance Tuning

## Baseline Checks

- CPU: `top`, `mpstat`
- Memory: `free`, `vmstat`
- Disk: `iostat`
- Network: `ss`, `sar -n`

## Rule

Measure first, tune second. Avoid kernel tuning without a reproducible bottleneck.
